# 投資情報収集AIエージェント 要件書

## 1. プロジェクト概要

各所に散らばっている株式投資関連情報を自動収集・整理し、毎朝Slackへレポートを投稿するAIエージェントを構築する。最終的にはWebダッシュボードも合わせて提供する。

---

## 2. 投資スタイル（コンテキスト）

| 市場 | スタイル | 重点情報 |
|------|----------|----------|
| 米国株 | 高配当株の積立 ＋ 値動き狙い | 配当利回り・増配履歴・決算・テーマ動向 |
| 日本株 | スイングトレード（短中期） | テクニカル・需給・ニュース・出来高 |

---

## 3. 解決したい課題

- 複数サイト・アプリに散らばった情報を毎日巡回する手間を削減する
- 重要ニュースの見逃しをなくす
- 「なぜ動いたか」の文脈を素早く把握する

---

## 4. 監視対象

- **個別銘柄**：日本株・米国株 合計30銘柄程度（ウォッチリストは設定ファイルで管理）
- **セクター動向**：
  - 米国：S&P500 セクター別ETF（XLK, XLF, XLE, XLV 等）
  - 日本：東証33業種インデックス

---

## 5. 収集する情報

### 5-1. 毎朝収集（必須）

| カテゴリ | 内容 |
|----------|------|
| 前日の値動き | 保有・ウォッチ銘柄の終値・騰落率・出来高 |
| 米国市場サマリー | 主要指数（S&P500, NASDAQ, DOW）の前日終値・騰落 |
| 日本市場前日サマリー | 日経225・TOPIX・マザーズ |
| セクター動向 | 米国・日本セクター別の前日騰落ランキング |
| 個別銘柄ニュース | ウォッチリスト銘柄に関するニュース（直近24時間） |
| 経済指標カレンダー | 当日・翌日の重要指標発表予定（FOMC, CPI, 雇用統計等） |
| 為替 | USD/JPY, EUR/USD 直近レート |

### 5-2. アラート（随時・将来対応）

- 出来高急増検知
- 急騰・急落アラート（±5%以上）
- 決算発表の直前通知

---

## 6. アウトプット仕様

### 6-1. Slack レポート（フェーズ1・最優先）

- **投稿タイミング**：毎朝6:00（日本時間）
- **チャンネル構成（推奨）**：
  - `#morning-report`：毎朝の定時レポート
  - `#alerts`：随時アラート（将来）
- **レポート構成**：
  1. 📊 市場サマリー（指数・為替）
  2. 🗓 本日の経済指標予定
  3. 📈 セクター動向（上位・下位3セクター）
  4. 🔍 ウォッチ銘柄ハイライト（動きのあった銘柄を優先）
  5. 📰 注目ニュース（銘柄別に整理）
  6. 💡 今日の注目ポイント（Claude による3行要約）

### 6-2. Webダッシュボード（フェーズ2）

- ウォッチリスト銘柄の一覧表示（価格・騰落率・チャート）
- セクターヒートマップ
- ニュースフィード（銘柄別フィルタ）
- 経済指標カレンダー

---

## 7. システム構成（技術スタック）

```
【データ収集】
  yfinance（株価・指数・ETF）
  NewsAPI or RSS（ニュース収集）
  investing.com / 株探（日本株ニュース）

【エージェント本体】
  Python 3.11+
  Claude API（ニュース要約・注目ポイント生成）
  APScheduler or cron（定時実行）

【通知】
  Slack Incoming Webhook

【ダッシュボード（フェーズ2）】
  Next.js / React
  Recharts（チャート）
  Vercel（ホスティング）
  Supabase（データ永続化）
```
